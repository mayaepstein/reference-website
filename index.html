Week 3: Naming Conventions & Commit Messages

Rules for naming files and paths

  • All lowercase
  • No spaces
  • Only letters, numbers, and dashes (-)

Commit Messages

Commit messages should be between 3-10 words. The should have proper grammar and spelling, a capital starting letter, and no periods. You should pretend that every commit starts with "This commit will..." and then add an imperative verb like create, fix, add, or solve.

Week 5: Structuring Content, Linking & Comments

Structure Elements

The tags used to structure content are:

  • header - introductory content
  • footer - appears at the bottom of a website
  • main - for the main page content, can only be used once per page
  • section - used to group related content
  • article - for self contained elements on a page
  • nav - contains navigation links

Linking on the Same Page

To link to an element on the same page, use an href tag and its class or id.

Week 6: Accessibility

Landmark Roles

Landmark roles help people with screen readers jump to specific content. They include:

  • header role: banner
  • nav role: navigation
  • main role: main
  • footer role: contentinfo

Skip Links

Put skip links at the top of your page to link to specific areas like nav or main content

Week 10: Styling Text, Fonts & Links

Font properties

  • font-family: specifies font
  • font-size: specifies font size
  • font-weight: for setting a font's weight
  • font-style: for setting italics
  • line-height: specifies line height
  • font shorthand: shorthand for style, weight, size, line height and family

Text decorations

  • text-transform: changes case, like lowercase or uppercase
  • text-decoration: for lines, like underlines or overlines
  • text-align: changes alignment, like right left or center
  • text-indent: idents the first line of text
  • text-shadow: for creating a drop shadow

Links

You can apply css properties to links, and change their different states. I can style the default, visited, hover, and focus state of this link.

Adding fonts

You can add fonts to your website by selecting some on google fonts and copy the link area. Then, paste it into the head of your site.
